# h-component

## Getting Started

Install dependencies,

```bash
$ npm i
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Run test,

```bash
$ npm test
```

Build library via `father`,

```bash
$ npm run build
```
```bash
lerna init // 初始化lerna配置
lerna create // 创建lerna包
lerna link // 链接lerna包
lerna add  // 安装单个依赖
lerna bootstrap // 安装全局依赖并链接到所有子包
lerna clean // 清除node_modules
lerna list // 列出子包
```
